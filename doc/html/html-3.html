第3天：超链接和图像标签

咱们今天来学习两个重要的标签：超链接标签和图像标签。这两个标签在网页开发中非常常用，能让页面更加生动有趣。

超链接标签

超链接标签使用<a>标签来定义，其中的href属性指定了链接的目标地址。

            
                <!DOCTYPE html>
                <html lang="en">
                <head>
                    <meta charset="UTF-8">
                    <meta name="viewport" content="width=device-width, initial-scale=1.0">
                    <title>超链接标签</title>
                </head>
                <body>
                  <a href="https://www.saycode.cn">点我跳转页面</a>
                </body>
                </html>

图像标签

图像标签用于在网页中嵌入图片，并可以设置图片的属性，比如宽度、高度等。

        
            <!DOCTYPE html>
            <html lang="en">
            <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>div</title>
            </head>
            <body>
              <div>
                <img src="图片.jpg" alt="图片描述" />
              </div>
            </body>
            </html>
